Configuración paso a paso

Sigue estos pasos en orden. No empieces el siguiente hasta completar el anterior.

Contratar y activar Conecta HUB

Conecta HUB es el middleware certificado para SAGE 50 Cloud. Debes contratar el plan adecuado al volumen de Dentix (~10.000 SKUs).

# URLs del servicio Panel de cliente: https://app.conectahub.com Documentación: https://docs.conectahub.com Soporte técnico: soporte@conectahub.com # Plan recomendado para Dentix Plan: Business (hasta 15.000 SKUs) Precio: ~75€/mes Sincronizaciones: ilimitadas

Una vez contratado, accede al panel y crea un nuevo canal de integración: SAGE 50 Cloud ↔ WooCommerce.

Instalar el plugin Conecta HUB en WooCommerce

Desde el panel de WordPress:

# Ruta en WordPress wp-admin → Plugins → Añadir nuevo → Buscar "Conecta HUB" # O instalar manualmente 1. Descargar .zip desde app.conectahub.com/downloads 2. wp-admin → Plugins → Subir plugin → Seleccionar .zip 3. Activar plugin # Versión mínima requerida WooCommerce: ≥ 8.0 WordPress: ≥ 6.4 PHP: ≥ 8.1

Conectar SAGE 50 Cloud con Conecta HUB

En SAGE 50 Cloud debes autorizar el acceso a la API. Ve a Herramientas → Integraciones → API REST.

# Credenciales necesarias de SAGE 50 Cloud client_id: SAGE50_CLIENT_ID (desde panel SAGE) client_secret: SAGE50_CLIENT_SECRET empresa_id: ID de la empresa Dentix en SAGE entorno: produccion | sandbox # Introducir en Conecta HUB app.conectahub.com → Mi canal → Configurar SAGE 50 → Pegar client_id y client_secret → Seleccionar empresa: "Dentix Productos Dentales S.L." → Autorizar y verificar conexión

Si la conexión es correcta, verás un indicador verde "SAGE 50 conectado" en el panel de Conecta HUB.

Conectar WooCommerce con Conecta HUB

# Generar clave API en WooCommerce wp-admin → WooCommerce → Ajustes → Avanzado → API REST → Añadir clave → Descripción: "Conecta HUB" → Permisos: Lectura/Escritura → Generar clave # Copiar en Conecta HUB consumer_key: ck_xxxxxxxxxxxxxxxxxxxxxxxxxx consumer_secret: cs_xxxxxxxxxxxxxxxxxxxxxxxxxx url_tienda: https://www.dentix.es # Webhook URL (copiar en WooCommerce) webhook: https://app.conectahub.com/webhook/TU_CANAL_ID

Registrar el webhook en: wp-admin → WooCommerce → Ajustes → Avanzado → Webhooks → Añadir webhook para los eventos: order.created, order.updated, customer.created.

Sincronización inicial: importar el catálogo completo

La primera vez, debes importar los ~10.000 SKUs de SAGE a WooCommerce. Esto puede tardar entre 2 y 6 horas según la conexión.

# En Conecta HUB → Mi canal → Sincronización manual 1. Seleccionar: "Importación inicial completa" 2. Elegir empresa: Dentix Productos Dentales S.L. 3. Marcar: SKU, precios, stock, imágenes, categorías 4. Ejecutar en horario de baja carga (noche) 5. Revisar log de errores tras finalizar # Comandos WP-CLI útiles para verificar wp wc product list --fields=id,sku,stock_quantity --format=csv wp wc product list --status=publish | wc -l # contar publicados wp cache flush # limpiar caché

Verificación: Comprueba que el número de productos publicados en WooCommerce coincide con los artículos activos en SAGE 50.

Monitorización y gestión de errores

Conecta HUB tiene un panel de logs en tiempo real. Además, configura alertas por email.

# Panel de logs Conecta HUB app.conectahub.com → Mi canal → Logs # Errores más comunes y solución Error 401: Token SAGE expirado → Reconectar en panel Conecta HUB Error 404: SKU no encontrado en WooCommerce → Importar artículo manualmente Error 422: Datos inválidos (precio vacío, EAN duplicado) → Revisar artículo en SAGE Stock 0: Artículo sin stock → WooCommerce lo marca "agotado" automáticamente Timeout: Servidor lento → Verificar VPS y aumentar max_execution_time a 120s # Alertas por email (configurar en Conecta HUB) Notificar cuando: error de sincronización, pedido no enviado a SAGE, cliente nuevo sin CIF, más de 10 errores en 1h

También activa las notificaciones nativas de WooCommerce para bajo stock en wp-admin → WooCommerce → Ajustes → Productos → Inventario.

Preguntas frecuentes técnicas

¿Qué pasa si modifico un producto en WooCommerce?
La próxima sincronización lo sobreescribirá con los datos de SAGE 50. Nunca modifiques catálogo en WooCommerce directamente. Hazlo siempre en SAGE.
¿Puedo tener productos solo en WooCommerce?
Sí. Los productos creados directamente en WooCommerce (sin SKU de SAGE) no son afectados por la sincronización. Útil para productos digitales o servicios.
¿Cómo activar la sincronización bidireccional?
Cuando Dentix esté lista, en Conecta HUB → Mi canal → Configuración avanzada → activar "Sincronización bidireccional". No requiere cambios de código.
¿Qué ocurre con los pedidos durante una caída de SAGE?
Los pedidos siguen procesándose en WooCommerce. Conecta HUB los encola y los reenvía a SAGE cuando la conexión se restaure.
¿Cómo se gestionan las tarifas especiales B2B?
Define tarifas por cliente en SAGE 50. Conecta HUB sincroniza los precios de cada tarifa con el rol de cliente correspondiente en WooCommerce.
¿Hay forma de probar la integración sin afectar producción?
Sí. Usa el entorno sandbox de Conecta HUB junto con la instancia InstaWP (rama pruebas de GitHub) para pruebas completas sin tocar el VPS de producción.
